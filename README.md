# GeneticNeuralNetwork
In this solution we try to use Genetic Algorithms (GA) to choose Artificial Neural Networks initialization weights in a smart way than just randomize it.

## Requirements:
* Python (>= 2.6 or >= 3.3),
* NumPy (>= 1.6.1),
* SciPy (>= 0.9),
* scikit-learn (>=0.18.1).
